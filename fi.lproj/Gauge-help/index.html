Laatan määrittely:

#[laatan_ominaisuudet] <liikennepaikat> <ratayhteydet> ”painettu teksti”

missä

  • [laatan_ominaisuudet] : pilkuilla erotettu määrittely: koodi,[värikoodi]<(erillisten_asentojen_lukumäärä)>
  • <liikennepaikat> : pilkuilla erotettu luettelo: [tyyppikoodi]<(arvo)><asemaympyröiden_lukumäärä>
  • <ratayhteydet> : pilkuilla erotettu luettelo: [lähtötaho_tai_liikennepaikka][raideväli][maalitaho_tai_liikennepaikka]
  • [värikoodi] : F =tyhjä, Y =keltainen, E =vihreä, R =punaruskea, G =harmaa (tai yhdistelmä näistä)
  • [tyyppikoodi] : C =kaupunki, V =kylä, O =etäkohde, J =vaihde
  • [lähtö/maalitaho_tai_liikennepaikka] : 0..5 =laatan taho (lounas..kaakko), S0..Sn indeksi edellä annettuun liikennepaikkojen luetteloon
  • [raideväli] : - =tavallinen, = =kapearaiteinen, + =kaksoisleveys
