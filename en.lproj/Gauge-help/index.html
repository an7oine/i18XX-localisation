Tile definition structure:

#[tile_properties] <location_definitions> <connection_definitions> ”imprinted text”

where

  • [tile_properties] : comma-separated list containing: code,[colour_abbreviation]<(number_of_distinct_orientations)>
  • <location_definitions> : comma-separated list of definitions: [type_abbreviation]<(value)><number_of_slots>
  • <connection_definitions> : comma-separated list of definitions: [starting_edge_or_location][gauge_of_track][ending_edge_or_location]
  • [colour_abbreviation] : (a combination of) F for buff, Y for yellow, E for green, R for russet, G for grey
  • [type_abbreviation] : C for city, V for village, O for offboard, J for junction
  • [starting/ending_edge_or_location] : 0..5 for edges SW..SE, S0..Sn for an index to intra-tile locations given above
  • [gauge_of_track] : - for regular, = for narrow, + for dual
